# Nero Chau | Personal Portfolio

A personal portfolio showcasing work at the intersection of **Computer Science, Mathematics, Data Science and Social Impact**. This site features interactive elements, a project filtering system, and smooth scroll animations.

Check out portfolio here: [Samphasnearyroth (Nero) Chau](https://nerochau.github.io/portfolio-nerochau/)

## Features

- **Typing Animation:** Dynamic hero section header that introduces key focuses.
- **Year-based Project Filter:** Custom JavaScript logic to sort projects by their completion year (2024, 2025, 2026).
- **Responsive Layout:** Optimized for all devices using CSS Flexbox and Grid.
- **Smooth Navigation:** Clean, accessible internal linking for a seamless user experience.

## Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling with Google Fonts (Nunito) and advanced layouts.
- **JavaScript (ES6):** DOM manipulation, animation triggers, and filtering logic.

## Featured Projects

- **Granted (2026):** AI Agent built with LangGraph to assist nonprofits in grant writing.
- **Career Matching ML (2025):** Python-based machine learning for job performance prediction.
- **National Exam Analysis (2025):** Data analysis using R to evaluate educational success predictors.
- **Choosr (2025):** A Chrome extension for consumer decision-making.
